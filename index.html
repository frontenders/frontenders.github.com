Frontenders es una reunión mensual centrada en temas de desarrollo frontend, diseño web, UI y UX. Nos encontramos regularmente el segundo jueves de cada mes en el espacio de coworking Workether, en Calle Serpis 3 (Valencia). ¿Quieres saber más?

Próximo meetup: jueves 10 abril 2014, 19:00h

Como inspeccionar Sass en Chrome usando su Workspace

Veremos como usar el Workspace de Chrome Dev Tools para guardar al vuelo los cambios mientras inspeccionamos el DOM y para los que usamos un preoprocesador CSS, modificar el archivo fuente y guardar los cambios para refrescarlos en pantalla automaticamente.

CSS Técnicas avanzadas

Es una exposición de problemas de diseño que pueden resolverse fácilmente con CSS, sin uso de JS. Veremos ejemplos prácticos de menor a mayor complejidad, además de un repaso sobre cómo personalizar elementos de formulario.

Finalmente se hará un pequeño resumen sobre propiedades no muy conocidas de CSS.
